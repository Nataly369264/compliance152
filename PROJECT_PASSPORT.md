
PROJECT PASSPORT — Compliance 152-ФЗ
Суть проекта
AI-сканер сайтов на соответствие требованиям 152-ФЗ «О персональных данных». Автоматически проверяет сайт, выявляет нарушения, рассчитывает штрафы по КоАП и генерирует пакет документов для устранения нарушений.

Целевая аудитория
Компании, которые собирают персональные данные на сайте и хотят проверить соответствие 152-ФЗ без дорогостоящего юридического аудита.

Ценностное предложение
За 1–2 минуты получить: score соответствия, список конкретных нарушений, расчёт потенциального штрафа и готовые документы для исправления ситуации.

Архитектура
URL сайта
    ↓
Краулер (crawler.py) — обходит страницы, ищет формы, 
    скрипты, cookie-баннеры, политику конфиденциальности
    ↓
Анализатор (analyzer.py) — проверяет 35+ пунктов чеклиста,
    считает score и штрафы по КоАП
    ↓
LLM (OpenRouter) — формирует текстовый анализ с рекомендациями
    ↓
Генератор (generator.py) — создаёт пакет документов из 12 шаблонов
    ↓
Web UI (/check) — отображает результат пользователю

Паттерны UI (зафиксированы 2026-03-01)
Клиентская анимация без библиотек (app.js):

countUp(el, target, duration) — requestAnimationFrame + easeOutCubic (1 - Math.pow(1-p, 3)), значения через Math.round()

Цвет/классы на элементе применяются синхронно ДО вызова rAF — гарантирует что DOM готов к первому кадру

Фазовый прогресс загрузки — _phaseTimers[], startPhaseProgress() / stopPhaseProgress():
  Фаза 1 «Сканирование страниц...» — сразу (0 сек)
  Фаза 2 «Анализ нарушений...» — setTimeout 8 000 мс
  Фаза 3 «Формируем отчёт...» — setTimeout 20 000 мс, ∞ до ответа

stopPhaseProgress() всегда вызывается в finally — мгновенный стоп независимо от исхода fetch

el.firstChild.textContent — меняет только текстовый узел, не трогает дочерние теги (<br>, <small>)
